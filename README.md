# spotifyProject
This project contains an end to end data engineering pipeline built around a spotify style music streaming analytics use case.
It demonstrated how raw data can be ingested, processed, transformed, modeled using Azure cloud services and Databricks.

# tech stack:
1. ADF
2. Azure Databricks
3. Azure SQL server
4. DLT
5. Pyspark
6. Azure data lake
7. spark structured streaming
